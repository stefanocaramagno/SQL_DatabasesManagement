# SQL (Databases Management)

**Hello everyone!**

My name is [Stefano Caramagno](https://www.linkedin.com/in/stefanocaramagno), and I'm pleased to present this repository containing exercises on **databases management** implemented in **SQL programming**. <br>
These exercises were completed as part of the **Database** course during my [Bachelor's Degree in Computer Engineering](https://www.dieei.unict.it/corsi/l-8-inf) at the [University of Catania](https://www.unict.it).

## üåç Context

This repository hosts a collection of SQL exercises designed to enhance proficiency in database management systems. <br>
The exercises cover a wide range of SQL concepts and advanced database operations, including the creation and use of views, triggers for enforcing data integrity, transactions for ensuring data consistency, and procedures for encapsulating SQL logic, specifically using MySQL.

## üéØ Objectives

- **Advanced SQL Querying**: Develop advanced SQL querying skills through exercises that involve complex joins, subqueries, and aggregate functions.
- **Database Management Skills**: Gain proficiency in database management tasks such as creating and managing views, triggers, transactions, and procedures.
- **Practical Applications**: Apply SQL concepts to solve practical database management challenges, emphasizing efficiency and performance.

## üìÇ Contents

- **Views**: Exercises focusing on creating, modifying, and utilizing views to simplify complex queries in MySQL.
- **Triggers**: Exercises demonstrating the use of triggers for enforcing data integrity constraints and automating database actions in MySQL.
- **Transactions**: Exercises covering transaction management to ensure data consistency and integrity in multi-operation scenarios using MySQL.
- **Procedures**: Exercises on creating and utilizing stored procedures to encapsulate and execute SQL logic efficiently in MySQL.
